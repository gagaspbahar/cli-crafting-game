## How to Use
---
### Dependencies
- C++
- ??

### Installation
```
git clone https://github.com/gagaspbahar/cli-crafting-game.git
```

### Running the Makefile
Make sure you are using Linux with C++17. WSL also works if you are using Windows.
```
sudo apt-get install make g++
make all
```
